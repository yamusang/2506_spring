타임리프 라이브러리 추가하면 resources 아래 static, template 폴더 생성됩니다.

  • static 폴더 : css, js, 이미지, 정적 html 등 정적 리소스를 저장하기
  • template 폴더 : 타임리프가 html 웹페이지를 만들기 위해 동적인 데이터를 활용합니다. 그 때 사용되는 html은 템플릿이라고 부르며 이 폴더에 저장합니다. 임의로 하위 폴더 생성 가능.

스프링 부트 웹 애플리케이션은 root context (http://localhost:8080/ 에서 마지막 /) 와 연결되는 view는 index.html(홈,인덱스 화면) 입니다.

예제 URL

  • /login
  • /join
  • /list
  • customer/list(애트리뷰트 테스트)
  • product/list(모델애트리뷰트 테스트)
  • product/searchAll(동적쿼리)
  • /getSessionAll 테스트
